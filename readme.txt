== Description ==

This plugin tracks each post hit by user. Admin will be able see a report of all the views in given period of time. Posts list will show Hits count against each of the posts. You can use the most viewed widget area to highlight most viewed posts of your site.

This plugin is free to use and is not actively supported by the author, but will be monitored for serious bugs that may need corrections.

Support and customization of this plugin provided by <a href="https://club.wpeka.com/">WPeka Club</a>. WPeka Club is a membership that gives you access to 85+ premium WordPress plugins and themes.

**Requirements:** WordPress 4.0+, PHP5.3 and higher

== Installation ==

1. Upload the "post-hit-stats" folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can see **Post Hit Stats** Setting in admin menu.
4. You can also set the Post Hit Stats widget
